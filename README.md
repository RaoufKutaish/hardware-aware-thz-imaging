# Hardware-Aware 3D Terahertz Imaging

This repository contains Python code associated with the manuscript:

**Hardware-Aware Sparse Terahertz Imaging: Single-Aperture Depth Estimation and Multi-Aperture 3D Reconstruction**

The code supports sparse and regularized reconstruction for computational terahertz (THz) imaging, including LASSO, elastic net, ridge regression, OMP, CoSaMP, and operator-aware unrolled LASSO.

## Repository Structure

```text
src/        Core reconstruction methods and utility functions
scripts/    Executable scripts for running experiments
examples/   Example notebooks and small demo data
docs/       Documentation and data descriptions
```
